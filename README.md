# FANCY QUICK PHONE BOOK

## FEATURES
1. Add users into phone book.
2. Search users by first name or last name in phone book.

## Tech Stack
React.js, react-redux, ReduxToolkit Typescript, react/testinglibrary, tailwindcss, postcss.

## Scripts
1. `yarn start`
2. `yarn build`
3. `yarn test`
